# To-Do List Manager

A simple command-line based To-Do List Manager written in Python. 
This program allows you to create, view, mark as completed, and delete tasks from a to-do list.

## Features

- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks

## Requirements

- Python Compiler
